# F-020 戶外遊憩設施

> 水利用地或農牧用地設置戶外遊憩設施，以自然景觀為基礎，開發成本相對較低。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 800 萬元/公頃 | |
| 年收益 | 300 萬元/公頃 | |
| 年維運成本 | 80 萬元/公頃 | |
| 使用年限 | 15 年 | |
| 折現率 | 5% | |
| 殘值率 | 10% | |
| 備註 | 含戶外運動場、野餐區；自然景觀加持效益大 | |

## 財務試算結果
