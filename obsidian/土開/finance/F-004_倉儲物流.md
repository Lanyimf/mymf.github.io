# F-004 倉儲物流

> 丁種建築用地設置倉儲物流設施，需重視聯外道路條件。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 坪 | |
| 建設成本 | 10 萬元/坪 | |
| 年收益 | 2 萬元/坪 | |
| 年維運成本 | 0.3 萬元/坪 | |
| 使用年限 | 30 年 | |
| 折現率 | 5% | |
| 殘值率 | 15% | |
| 備註 | 近道路優先 | |

## 財務試算結果

| 指標 | 數值 | 說明 |
|------|------|------|
| **ROI（年報酬率）** | **17.0%** | (年收益－維運成本) / 建設成本 |
| **回收年限** | **5.9 年** | 建設成本 / 年淨收益 |
| **NPV（30年）** | **16 萬元** | 折現率 5%，含殘值 |

## 試算公式

```
年淨收益　= 年收益 - 年維運成本
         = 2 - 0.3 = 1.7 萬元

ROI　　　 = 1.7 / 10 × 100 = 17.0%

回收年限　= 10 / 1.7 = 5.9 年

NPV　　　 = Σ(年淨收益 × 折現因子) + 殘值現值 - 建設成本
         = 16 萬元（30年期，折現率5%）
```

## 場址套入試算

> 將實際場址面積代入以下公式：

| 欄位 | 公式 |
|------|------|
| 實際建設成本 | `場址面積（坪）× 10 萬元` |
| 實際年收益 | `場址面積（坪）× 2 萬元` |
| 實際年維運 | `場址面積（坪）× 0.3 萬元` |
