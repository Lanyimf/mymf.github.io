# F-005 休閒農場

> 農牧用地設置休閒農場，需依休閒農業輔導管理辦法申請籌設許可。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 600 萬元/公頃 | |
| 年收益 | 200 萬元/公頃 | |
| 年維運成本 | 60 萬元/公頃 | |
| 使用年限 | 15 年 | |
| 折現率 | 5% | |
| 殘值率 | 10% | |
| 備註 | 最小面積0.5公頃 | |

## 財務試算結果

| 指標 | 數值 | 說明 |
|------|------|------|
| **ROI（年報酬率）** | **23.3%** | (年收益－維運成本) / 建設成本 |
| **回收年限** | **4.3 年** | 建設成本 / 年淨收益 |
| **NPV（15年）** | **882 萬元** | 折現率 5%，含殘值 |

## 試算公式

```
年淨收益　= 年收益 - 年維運成本
         = 200 - 60 = 140 萬元

ROI　　　 = 140 / 600 × 100 = 23.3%

回收年限　= 600 / 140 = 4.3 年

NPV　　　 = Σ(年淨收益 × 折現因子) + 殘值現值 - 建設成本
         = 882 萬元（15年期，折現率5%）
```
