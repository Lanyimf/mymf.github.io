# F-014 住宅（乙種建築用地）

> 乙種建築用地興建住宅，需符合建築法建造執照申請規定，供鄉村地區居住使用。

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 坪 | |
| 建設成本 | 15 萬元/坪 | |
| 年收益 | 0.6 萬元/坪 | |
| 年維運成本 | 0.1 萬元/坪 | |
| 使用年限 | 50 年 | |
| 折現率 | 3% | |
| 殘值率 | 30% | |
| 備註 | 鄉村地區住宅；月租以2,000元/坪估算 | |

## 風險提醒

- 鄉村地區租金收益有限
- 需符合建蔽率、容積率限制
- 污染場址上建築有健康風險，需整治完成
