# F-017 無公害性小型工業設施

> 乙種建築用地設置無公害性小型工業設施，適合低污染、低噪音的輕工業或手工業。

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 坪 | |
| 建設成本 | 10 萬元/坪 | |
| 年收益 | 2.5 萬元/坪 | |
| 年維運成本 | 0.4 萬元/坪 | |
| 使用年限 | 20 年 | |
| 折現率 | 5% | |
| 殘值率 | 15% | |
| 備註 | 低污染製造業；乙種建築用地適用 | |
