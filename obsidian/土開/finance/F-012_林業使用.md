# F-012 林業使用(造林、苗圃)

> 農牧用地造林或設置苗圃，收益長期但穩定。政府有造林補貼計畫可降低初期投入。

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 30 萬元/公頃 | |
| 年收益 | 5 萬元/公頃 | |
| 年維運成本 | 2 萬元/公頃 | |
| 使用年限 | 30 年 | |
| 折現率 | 3% | |
| 殘值率 | 50% | |
| 備註 | 造林補貼可提升收益；木材採伐為長期投資 | |
