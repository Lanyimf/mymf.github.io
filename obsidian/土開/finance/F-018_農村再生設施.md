# F-018 農村再生設施

> 農牧用地設置農村再生設施，以活化農村聚落為目的，可結合政府農村再生計畫申請補助。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 500 萬元/公頃 | |
| 年收益 | 100 萬元/公頃 | |
| 年維運成本 | 30 萬元/公頃 | |
| 使用年限 | 20 年 | |
| 折現率 | 4% | |
| 殘值率 | 10% | |
| 備註 | 含農村文化館、社區廣場；部分可申請農村再生基金補助 | |
