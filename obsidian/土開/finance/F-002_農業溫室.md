# F-002 農業溫室

> 農牧用地設置溫室農業設施，需符合農業設施容許使用規定。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 800 萬元/公頃 | |
| 年收益 | 120 萬元/公頃 | |
| 年維運成本 | 30 萬元/公頃 | |
| 使用年限 | 15 年 | |
| 折現率 | 5% | |
| 殘值率 | 10% | |
| 備註 | 含水電設備 | |

## 財務試算結果

| 指標 | 數值 | 說明 |
|------|------|------|
| **ROI（年報酬率）** | **11.2%** | (年收益－維運成本) / 建設成本 |
| **回收年限** | **8.9 年** | 建設成本 / 年淨收益 |
| **NPV（15年）** | **173 萬元** | 折現率 5%，含殘值 |

## 試算公式

```
年淨收益　= 年收益 - 年維運成本
         = 120 - 30 = 90 萬元

ROI　　　 = 90 / 800 × 100 = 11.2%

回收年限　= 800 / 90 = 8.9 年

NPV　　　 = Σ(年淨收益 × 折現因子) + 殘值現值 - 建設成本
         = 173 萬元（15年期，折現率5%）
```
