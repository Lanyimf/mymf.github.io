# F-003 工業廠房

> 丁種建築用地設置工業廠房，需符合建築法及工廠管理輔導法規定。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 坪 | |
| 建設成本 | 15 萬元/坪 | |
| 年收益 | 3 萬元/坪 | |
| 年維運成本 | 0.5 萬元/坪 | |
| 使用年限 | 30 年 | |
| 折現率 | 5% | |
| 殘值率 | 20% | |
| 備註 | 乙丁種建地適用 | |

## 財務試算結果

| 指標 | 數值 | 說明 |
|------|------|------|
| **ROI（年報酬率）** | **16.7%** | (年收益－維運成本) / 建設成本 |
| **回收年限** | **6.0 年** | 建設成本 / 年淨收益 |
| **NPV（30年）** | **24 萬元** | 折現率 5%，含殘值 |

## 試算公式

```
年淨收益　= 年收益 - 年維運成本
         = 3 - 0.5 = 2.5 萬元

ROI　　　 = 2.5 / 15 × 100 = 16.7%

回收年限　= 15 / 2.5 = 6.0 年

NPV　　　 = Σ(年淨收益 × 折現因子) + 殘值現值 - 建設成本
         = 24 萬元（30年期，折現率5%）
```

## 場址套入試算

> 將實際場址面積代入以下公式：

| 欄位 | 公式 |
|------|------|
| 實際建設成本 | `場址面積（坪）× 15 萬元` |
| 實際年收益 | `場址面積（坪）× 3 萬元` |
| 實際年維運 | `場址面積（坪）× 0.5 萬元` |

## 風險提醒

- 以上數值為參考基準，實際建設成本依地形、工程難度調整
