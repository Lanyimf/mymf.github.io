# F-006 殯葬設施

> 殯葬用地設置殯葬設施，需依殯葬管理條例取得設置許可，選址有距離限制。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 5,000 萬元/公頃 | |
| 年收益 | 800 萬元/公頃 | |
| 年維運成本 | 150 萬元/公頃 | |
| 使用年限 | 50 年 | |
| 折現率 | 5% | |
| 殘值率 | 30% | |
| 備註 | 含靈骨塔及公墓 | |

## 財務試算結果

| 指標 | 數值 | 說明 |
|------|------|------|
| **ROI（年報酬率）** | **13.0%** | (年收益－維運成本) / 建設成本 |
| **回收年限** | **7.7 年** | 建設成本 / 年淨收益 |
| **NPV（50年）** | **6,997 萬元** | 折現率 5%，含殘值 |

## 試算公式

```
年淨收益　= 年收益 - 年維運成本
         = 800 - 150 = 650 萬元

ROI　　　 = 650 / 5,000 × 100 = 13.0%

回收年限　= 5,000 / 650 = 7.7 年

NPV　　　 = Σ(年淨收益 × 折現因子) + 殘值現值 - 建設成本
         = 6,997 萬元（50年期，折現率5%）
```

## 場址套入試算

> 將實際場址面積代入以下公式：

| 欄位 | 公式 |
|------|------|
| 實際建設成本 | `場址面積（公頃）× 5,000 萬元` |
| 實際年收益 | `場址面積（公頃）× 800 萬元` |
| 實際年維運 | `場址面積（公頃）× 150 萬元` |
