# F-009 農作產銷設施

> 農牧用地設置農作產銷設施（冷藏、洗選、加工包裝等），可提升農產品附加價值。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | 公頃 | |
| 建設成本 | 300 萬元/公頃 | |
| 年收益 | 150 萬元/公頃 | |
| 年維運成本 | 40 萬元/公頃 | |
| 使用年限 | 15 年 | |
| 折現率 | 5% | |
| 殘值率 | 10% | |
| 備註 | 含冷藏庫、洗選包裝設備；設施面積不超過農地40% | |

## 財務試算結果

| 指標 | 數值 |
|------|------|
| **ROI（年報酬率）** | **36.7%** |
| **回收年限** | **2.7 年** |
| **NPV（15年）** | **856 萬元** |
