# F-001 地面型太陽能

> 農牧用地或乙種建築用地設置地面型太陽能，需確認台電饋線容量及農電共生規劃。

## 基本參數

| 項目 | 數值 | 說明 |
|------|------|------|
| 計算單位 | MW | |
| 建設成本 | 3,000 萬元/MW | |
| 年收益 | 400 萬元/MW | |
| 年維運成本 | 50 萬元/MW | |
| 使用年限 | 20 年 | |
| 折現率 | 5% | |
| 殘值率 | 5% | |
| 備註 | 1MW約1公頃，FIT約4元/度 | |

## 財務試算結果

| 指標 | 數值 | 說明 |
|------|------|------|
| **ROI（年報酬率）** | **11.7%** | (年收益－維運成本) / 建設成本 |
| **回收年限** | **8.6 年** | 建設成本 / 年淨收益 |
| **NPV（20年）** | **1,418 萬元** | 折現率 5%，含殘值 |

## 試算公式

```
年淨收益　= 年收益 - 年維運成本
         = 400 - 50 = 350 萬元

ROI　　　 = 350 / 3,000 × 100 = 11.7%

回收年限　= 3,000 / 350 = 8.6 年

NPV　　　 = Σ(年淨收益 × 折現因子) + 殘值現值 - 建設成本
         = 1,418 萬元（20年期，折現率5%）
```

| 實際年維運 | `場址面積（MW）× 50 萬元` |
